## Overview
Summarizes the code when using the usual Moca.NET Framework, I have served as snippets.  
supports VB.NET and C# code.  

## Snippets
### Moca.NET DataAccessObject
* DAODelete
* DAOInsert
* DAOSelect
* DAOStoredPrepare
* DAOStoredUpdate
* DAOUpdate
### Log4net
* Log4net
### Web
* ICookie
* IViewStat
